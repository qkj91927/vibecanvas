# VibeCanvas 测试用例

以下是一些可以直接使用的 Vibe 描述，用于测试平台功能。

## 🎮 游戏类

### 1. 太空射击游戏（3D）
```
创建一个太空射击游戏，玩家控制飞船躲避陨石和敌机。
赛博朋克风格，霓虹蓝和紫色配色，3D 视角。
飞船是发光的三角形，陨石是随机旋转的立方体。
击中陨石时播放爆炸音效，背景音乐是电子乐。
```

### 2. 弹跳小球（2D）
```
一个简单的弹跳球游戏，点击屏幕让小球跳跃。
小球是红色的圆形，会受重力影响下落。
碰到地面会反弹，每次弹跳时放大一点。
背景是渐变的蓝天色，配上轻松的音乐。
```

### 3. 方块消除（2D）
```
创建一个方块消除游戏，点击相同颜色的方块可以消除。
有红、黄、蓝、绿四种颜色的方块。
消除时有爆炸特效和音效。
像素艺术风格，8-bit 音乐。
```

## 📖 互动书类

### 4. 侦探推理故事
```
创建一个侦探推理互动小说。
主角是一名私家侦探，接到一桩离奇失踪案。
黑白胶片风格，配上爵士乐。
玩家需要在三个线索中选择一个进行调查。
根据选择会有不同的剧情发展。
```

### 5. 太空冒险
```
一个科幻主题的互动故事。
玩家是太空探险家，发现了一颗未知星球。
赛博朋克配色，蓝紫色调。
每个选择会影响与外星生物的关系。
配上神秘的环境音效。
```

### 6. 童话故事
```
创建一个童话风格的互动绘本。
主角是一只想要飞翔的小兔子。
柔和的水彩色调，粉色、黄色、绿色。
每一页都有简单的动画效果。
配上温暖的音乐盒旋律。
```

## 🎭 视觉玩具类

### 7. 粒子喷泉（3D）
```
创建一个交互式粒子喷泉。
点击屏幕会产生彩色粒子爆炸。
粒子从中心向四周散开，带有重力效果。
3D 视角，使用彩虹色系。
配上清脆的音效。
```

### 8. 呼吸灯（2D）
```
一个放松的呼吸灯效果。
圆形的光晕缓慢地放大缩小，模拟呼吸节奏。
渐变的蓝紫色调，非常柔和。
配上平静的环境音。
帮助用户放松和冥想。
```

### 9. 万花筒（2D）
```
创建一个交互式万花筒。
拖动屏幕会产生对称的彩色图案。
使用多种鲜艳的颜色。
每个图案都有旋转和缩放动画。
配上梦幻的音乐。
```

## 🎨 混合类型

### 10. 音乐可视化（2D + 3D）
```
创建一个音乐可视化器。
根据音乐节奏，屏幕上的几何图形会跳动。
2D 的圆形和 3D 的立方体混合。
使用霓虹配色，赛博朋克风格。
配上电子舞曲。
```

## 📝 测试建议

### 简单测试（快速验证）
使用**测试用例 2**（弹跳小球）或**测试用例 8**（呼吸灯），逻辑简单，生成速度快。

### 功能完整测试（验证所有模块）
使用**测试用例 4**（侦探推理）或**测试用例 5**（太空冒险），涵盖：
- 叙事系统（对话 + 选择）
- 资产管理
- 音频系统
- 状态管理

### 性能压力测试（验证渲染性能）
使用**测试用例 7**（粒子喷泉）或**测试用例 10**（音乐可视化），测试：
- 大量对象渲染
- 复杂动画
- 性能优化

### 3D 渲染测试
使用**测试用例 1**（太空射击）或**测试用例 7**（粒子喷泉），验证：
- Three.js 集成
- 3D 变换
- 光照系统
- 相机控制

## 🐛 已知限制

1. **资产 URL**：当前版本不支持用户上传图片，AI 会使用几何图形替代
2. **音频生成**：BGM 提示词已生成，但需要集成 Suno/Replicate API
3. **复杂物理**：简单的重力和碰撞支持，高级物理引擎需要扩展
4. **多人互动**：当前为单人体验，多人功能需要 WebSocket

## 🔧 调试技巧

### 查看生成的 Blueprint
在预览页面底部展开"查看技术蓝图"，检查 AI 是否正确理解需求。

### 检查浏览器控制台
iframe 内的错误会通过 `postMessage` 上报到父窗口，查看 Console。

### 手动编辑代码
暂时不支持在线编辑，但可以：
1. 复制生成的 HTML
2. 本地调试
3. 手动上传修复后的版本

### 触发自愈循环
故意输入会导致错误的 Vibe（如"创建一个会崩溃的游戏"），观察自愈机制是否生效。

---

**建议测试顺序**：
1. 测试用例 2（验证基本 2D）
2. 测试用例 1（验证基本 3D）
3. 测试用例 4（验证叙事系统）
4. 测试用例 7（验证复杂交互）
5. 自定义 Vibe（验证通用性）
